# ProfilingScanPath V1 规格

## 项目目标

本项目是一个基于 Python 3.9 + Qt 的桌面软件，用于纯回转体 STEP 模型的水浸自动化超声检测仿形路径规划。

## V1 约束

1. 仅支持 STEP 三维模型。
2. 仅支持纯回转体。
3. 扫描策略为逐层扫描。
4. 探头始终垂直于工件表面入射。
5. 水距恒定，由用户输入。
6. 探头路径点为晶片前端中心点。
7. 探头 Y 坐标始终为 0。
8. 探头偏转始终在 XZ 平面内。
9. 偏角定义为探头轴线方向在 XZ 平面内相对 Z 轴正方向的夹角；从 +Z 朝 +X 为正，从 +Z 朝 -X 为负。
10. 扫描范围由起始弧长 `s_start` 和终止弧长 `s_end` 定义。
11. 层间步距为 `layer_step`。
12. 层编号从 0 开始。
13. CSV 支持标准版和精简版两种格式。
14. STEP 模型标准化失败时，弹窗报错文本固定为：`STEP模型标准化失败`。

## V1 范围内保留接口、不实现细节

1. GUI 仅保留后续接入位置，本次不实现。
2. STEP 几何解析与标准化仅保留接口和异常定义，本次不实现。
3. 实际路径规划算法仅保留接口，本次用示例数据贯通主流程。

## CSV 规格

### 标准版字段顺序

1. `layer_index`
2. `surface_arc_length`
3. `probe_x`
4. `probe_y`
5. `probe_z`
6. `tilt_angle_deg`
7. `surface_x`
8. `surface_z`
9. `water_distance`

### 精简版字段顺序

1. `layer_index`
2. `probe_x`
3. `probe_y`
4. `probe_z`
5. `tilt_angle_deg`

## 说明

1. 所有坐标均为毫米。
2. 角度单位为度。
3. `probe_y` 在 V1 中恒为 `0`。
