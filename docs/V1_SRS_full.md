# 水浸自动化超声检测仿形路径规划软件 V1 完整需求规格书（SRS）

## 1. 项目概述

本软件用于基于 STEP 三维模型生成纯回转体工件的自动化超声检测扫描路径。

------------------------------------------------------------------------

## 2. 系统功能

### 2.1 输入

-   STEP 三维模型
-   参数：
    -   起始弧长 s_start
    -   终止弧长 s_end
    -   层间步距 layer_step
    -   水距 water_distance

### 2.2 输出

CSV（标准版 / 精简版）

------------------------------------------------------------------------

## 3. 坐标系统

-   原点：旋转轴底点
-   Z轴：旋转轴方向
-   X轴：右
-   Y轴：内

------------------------------------------------------------------------

## 4. 探头模型

-   Y=0
-   法向入射
-   固定水距
-   偏角在XZ平面

------------------------------------------------------------------------

## 5. 数学模型

### 5.1 母线

C(s) = (x(s), z(s))

### 5.2 层生成

s_i = s_start + i \* layer_step

### 5.3 法向计算

T = (dx/ds, dz/ds) N = (-dz, dx)

### 5.4 探头点

Q = P + dN

### 5.5 偏角

θ = atan2(nx, nz)

------------------------------------------------------------------------

## 6. 算法流程

1.  读取STEP
2.  标准化坐标
3.  提取母线
4.  计算弧长
5.  离散层
6.  计算法向
7.  生成探头点
8.  导出CSV

------------------------------------------------------------------------

## 7. 数据结构

class ScanParams: s_start: float s_end: float layer_step: float
water_distance: float

class PathPoint: layer_index: int arc_length: float probe_x: float
probe_y: float probe_z: float tilt_angle_deg: float

------------------------------------------------------------------------

### 8.2 显示区

------------------------------------------------------------------------

## 9. 异常处理

------------------------------------------------------------------------

## 10. 参数校验

-   layer_step \> 0
-   water_distance \> 0
-   s_start \< s_end
