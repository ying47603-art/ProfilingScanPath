# V2 阶段总结

## 1. 项目概述

- 项目目标：面向回转体工件的扫描路径规划，完成从 STEP 几何输入到母线提取、路径生成、预览与导出的完整桌面化工作流。
- 技术栈：Python + OCP + PyQt6 + Matplotlib + PyVista/PyVistaQt。

## 2. V2 实现范围

V2 在保留 V1 核心算法链路的基础上，重点完成了 GUI 工程化和预览能力增强。

- 保留并复用现有主链路：
  - STEP 读取
  - 标准化
  - 母线提取
  - 路径生成
  - CSV 导出
- Qt Designer 界面正式接入：
  - `.ui -> pyuic6 -> ui_main_window.py -> main_window.py`
- 2D 预览接入：
  - 显示母线
  - 显示路径点
  - 显示起点 / 终点
  - 显示坐标轴和网格
- 3D 预览接入：
  - 显示旋转轴
  - 显示母线 3D 线
  - 显示回转参考线框
  - 显示路径 3D 折线与点
- 3D 显示设置接入：
  - 母线显示开关
  - 路径显示开关
  - 线框显示开关
  - 表面显示开关
  - 坐标轴 / 旋转轴 / 标签显示开关
  - 渲染模式切换
  - 回转分段数
  - 表面透明度
  - 平滑着色
  - 自动适配视图
  - 重置视角
- 修正类按钮接入：
  - 翻转 Z 轴
  - 翻转母线
  - 重新标准化

## 3. 核心架构

- `core/`
  - STEP 读取、标准化、母线提取、路径规划等核心逻辑
- `data/`
  - `StepModel`、`ScanPath`、`PathPoint` 等数据模型
- `gui/`
  - 主窗口、控制器、Designer 生成代码、预览控件
- `gui/widgets/`
  - 2D 预览控件
  - 3D 预览控件
- `scripts/`
  - GUI 启动脚本
  - 调试与验证脚本
- `tests/`
  - 圆柱、圆台、圆角件与 OCP 链路测试

V2 继续保持“薄 GUI + 厚 core”的设计原则：

- GUI 负责：
  - 参数读取
  - 信号槽联动
  - 状态展示
  - 预览刷新
- core 负责：
  - 几何标准化
  - 母线提取
  - 路径生成
  - 导出逻辑

这样可以在不改算法定义的前提下持续增强桌面交互能力。

## 4. V2 关键能力

### 4.1 Designer 驱动 GUI

- 不再依赖纯手写界面布局
- UI 结构由 Designer 维护
- 运行逻辑由 `main_window.py` 绑定
- 便于后续继续扩展按钮、设置区与预览区

### 4.2 2D 预览

- 基于 Matplotlib 嵌入 PyQt6
- 显示 `profile_points`
- 显示 `scan_path(surface_x/surface_z)`
- 起点 / 终点标记与母线路径区分清晰
- 空状态下保留坐标轴、边框和网格

### 4.3 3D 预览

- 基于 PyVista + PyVistaQt
- 显示：
  - 母线 3D polyline
  - 旋转轴
  - 回转参考线框
  - `probe_x / probe_y / probe_z` 路径折线与点
- 已完成视觉层级优化：
  - 母线主线最强
  - 母线起终点最强强调
  - 路径折线次强
  - 路径普通点更弱
  - 回转线框作为背景参考

### 4.4 3D 显示设置

当前 3D 设置区已经可以真实控制 3D widget，包含：

- 显示母线
- 显示路径
- 显示回转框线
- 显示表面
- 显示坐标轴
- 显示旋转轴
- 显示标签
- 渲染模式
- 回转分段数
- 表面透明度
- 平滑着色
- 自动适配视图
- 重置视角

其中回转表面 mesh 完全在 3D widget 内部基于 `profile_points` 构建，没有修改 controller 和 core。

### 4.5 修正联动

以下操作已与预览和路径状态联动：

- 翻转 Z 轴
- 翻转母线
- 重新标准化

联动效果：

- 刷新母线统计
- 清空旧路径结果
- 刷新 2D / 3D 预览
- 提示旧路径已失效，需要重新生成

## 5. 已验证能力

- 圆柱：
  - 母线范围正确
  - 2D / 3D 预览正确
  - 角度接近 90°
- 圆台 / 锥体：
  - 母线提取正确
  - 2D / 3D 预览可清晰展示空间关系
- 圆角过渡件：
  - 母线连续
  - 路径连续
  - 预览可展示过渡趋势
- GUI 主链路：
  - 加载 STEP
  - 提取母线
  - 生成路径
  - 导出 CSV
  - 修正按钮联动
  - 2D / 3D 同步刷新

## 6. 当前已知限制

- 当前 3D 表面仅是基于母线生成的简化回转表面，不是 STEP 实体真实显示。
- 3D 预览主要用于工程辅助观察，不是完整 CAD 可视化替代品。
- 空状态提示文字已简化为无文本，仅保留坐标轴和参考背景。
- 当前中文字体显示仍可能受 Matplotlib / VTK 默认字体影响，但不影响主链路功能。
- 显示设置虽然已接通，但尚未外扩为更复杂的可持久化用户偏好配置。
- UI 仍在持续收敛阶段，Designer 布局与控件命名仍可能继续微调。

## 7. 与 V1 的主要区别

相较于 V1，V2 的核心提升不在算法重写，而在于“可操作性”和“可解释性”：

- 从最小原型 GUI 升级到 Designer 驱动 GUI
- 从纯结果输出升级到 2D / 3D 可视化
- 从单纯按钮链路升级到修正联动和状态管理
- 从静态显示升级到可控的 3D 显示设置

V2 使用户能够更直接地理解：

- 当前母线几何
- 当前路径分布
- 回转体空间关系
- 修正动作对几何与路径的影响
