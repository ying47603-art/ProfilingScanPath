# V2 GUI 使用说明

## 1. 适用范围

本文档面向当前 V2.0 图形界面使用者，说明如何通过 GUI 完成：

- 加载 STEP
- 提取母线
- 生成路径
- 预览 2D / 3D 结果
- 导出 CSV
- 使用基础修正功能
- 使用 3D 显示设置

## 3. 主界面区域说明

### 3.1 顶部主按钮

- `加载STEP`
  - 打开文件选择框
  - 选择 STEP 文件后立即执行加载
- `提取母线`
  - 对当前 STEP 进行标准化并提取母线
- `生成路径`
  - 根据当前母线和参数生成扫描路径
- `导出CSV`
  - 导出母线与路径结果

### 3.2 左侧参数与修正区

- 文件路径
  - 当前选中的 STEP 文件路径
- 参数设置
  - `samples`
  - `s_start`
  - `s_end`
  - `自动使用总弧长`
  - `layer_step`
  - `water_distance`
- 修正按钮
  - `翻转Z轴`
  - `翻转母线`
  - `重新标准化`
- 3D 显示设置
  - 显示母线
  - 显示路径
  - 显示回转框线
  - 显示表面
  - 显示坐标轴
  - 显示旋转轴
  - 显示标签
  - 渲染模式
  - 回转分段数
  - 表面透明度
  - 平滑着色
  - 自动适配视图
  - 重置视角

### 3.3 中间预览区

- 2D 视图
  - 显示母线
  - 显示路径点
  - 显示起点 / 终点
- 3D 视图
  - 显示旋转轴
  - 显示母线 3D 线
  - 显示回转参考线框
  - 显示路径 3D 点 / 折线
  - 显示可选的简化回转表面

### 3.4 底部日志区

日志用于显示当前操作结果和异常信息。

常见标签包括：

- `[STEP]`
- `[PROFILE]`
- `[PATH]`
- `[EXPORT]`
- `[ERROR]`

## 4. 标准操作流程

### 4.1 加载 STEP

1. 点击 `加载STEP`
2. 选择 `.step` 或 `.stp` 文件
3. 成功后：
   - 当前文件路径更新
   - 日志显示加载结果
   - 旧预览清空

### 4.2 提取母线

1. 设置或确认参数
2. 点击 `提取母线`
3. 成功后：
   - 2D 视图显示母线
   - 3D 视图显示母线、旋转轴、回转参考线
   - 日志显示点数与范围信息

### 4.3 生成路径

1. 确认：
   - `s_start`
   - `s_end`
   - `layer_step`
   - `water_distance`
2. 若勾选 `自动使用总弧长`
   - `s_end` 会自动取当前母线离散总弧长
3. 点击 `生成路径`
4. 成功后：
   - 2D 视图叠加路径点
   - 3D 视图叠加路径点与折线
   - 日志显示路径点数与角度范围

### 4.4 导出 CSV

1. 点击 `导出CSV`
2. 选择导出目录
3. 成功后会生成：
   - `profile_points.csv`
   - `scan_path_standard.csv`
   - `scan_path_compact.csv`

## 5. 修正功能说明

### 5.1 翻转 Z 轴

作用：
- 对当前母线做 Z 方向翻转

影响：
- 2D / 3D 预览立即刷新
- 已生成路径会失效
- 需要重新点击 `生成路径`

### 5.2 翻转母线

作用：
- 反转当前母线点序

影响：
- 起点 / 终点对调
- 2D / 3D 预览立即刷新
- 已生成路径会失效

### 5.3 重新标准化

作用：
- 重新对当前 STEP 执行标准化并重新提取母线

适用场景：
- 修改参数后希望重新得到母线
- 做过修正后希望回到重新提取状态

## 6. 3D 显示设置说明

### 6.1 显示开关

- `显示母线`
  - 控制母线主线与端点显示
- `显示路径`
  - 控制路径折线、路径点、路径端点显示
- `显示回转框线`
  - 控制回转体参考线框
- `显示表面`
  - 控制简化回转表面 mesh
- `显示坐标轴`
  - 控制 3D 坐标轴与网格
- `显示旋转轴`
  - 控制 Z 轴旋转轴线
- `显示标签`
  - 控制 Start / End 等辅助标签

### 6.2 渲染模式

- `线框`
  - 仅显示线框类对象
- `低精度表面`
  - 用较低分段数构造回转表面
- `高精度表面`
  - 用当前回转分段数构造更细致表面

### 6.3 参数设置

- `回转分段数`
  - 影响回转表面和平面参考线的细致程度
- `表面透明度`
  - 控制表面 mesh 透明度
- `平滑着色`
  - 控制表面显示质感
- `自动适配视图`
  - 开启后刷新数据时自动调整相机
  - 关闭后尽量保持当前手动视角

### 6.4 重置视角

- 点击 `重置视角`
- 恢复当前约定的默认工程视角
- 不会清空数据

## 7. 结果解读建议

### 7.1 2D 视图

适合观察：

- 母线几何形状
- 起点 / 终点方向
- 路径在母线上的覆盖趋势

### 7.2 3D 视图

适合观察：

- 旋转体空间关系
- 路径在 3D 空间中的分布
- 旋转轴与回转体的相对位置

## 8. 常见情况

### 8.1 提取后没有路径

这是正常的。

原因：
- `提取母线` 只生成母线
- 还没有执行 `生成路径`

### 8.2 修正后路径消失

这是正常设计。

原因：
- 翻转或重新标准化会改变母线
- 旧路径不再可信
- 需要重新生成路径

### 8.3 空视图只有坐标轴和背景

这是正常的空状态。

说明：
- 当前还没有可显示的母线或路径数据

## 9. 当前限制

- 当前 3D 表面为基于母线生成的简化回转表面，不是 STEP 实体真实显示
- 当前 3D 主要用于工程辅助观察，不是完整 CAD 模型查看器
- 2D / 3D 视图目前不提供复杂交互分析功能
- 中文字体显示效果受本机 Matplotlib / VTK 环境影响

## 10. 建议使用顺序

推荐每次按以下顺序操作：

1. 加载 STEP
2. 提取母线
3. 检查 2D / 3D 预览
4. 如有必要，执行翻转或重新标准化
5. 生成路径
6. 检查路径预览
7. 导出 CSV
